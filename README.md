Authors

Description
This matches the structure of the original blob sketch from Week 2 but moves details into classes.

This project is a 2D platformer built using p5.js that demonstrates how to organize a game using object oriented programming and JSON data. The structure is based on the original blob platformer from Week 2, but has been refactored so that game elements are separated into classes and level data is stored externally.

The player controls a blob character that can jump between platforms. Each level is defined using JSON, which allows the layout, gravity, and theme to be changed without modifying the core game logic. This approach makes the code more modular, readable, and scalable.

Learning Goals
Learning Goals:

load JSON in preload()
create WorldLevel from JSON
create BlobPlayer
update + draw each frame
handle input events (jump, optional next level)
